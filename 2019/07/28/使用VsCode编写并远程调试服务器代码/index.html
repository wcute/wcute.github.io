使用VsCode编写并远程调试服务器代码

Posted on 2019-07-28 | In Python

个人记录一下，虽然很简单但是网上文章还是踩了不少坑。
编辑器：VsCode
插件：remote Development
服务器：Linux（Centos）

安装要求

  • VsCode 客户端只需要安装remote Development插件即可
  • 服务器端必须要开启SSH服务，这个可以各自根据自己的服务器系统自行搭建SSH服务

搭建过程

首先安装remote Development插件，如下图搜索插件名称直接安装

安装完后，左下角会出现一个绿色的终端小框

点击小框，弹出一个框，选择连接到主机（Remote-SSH:Connect to Host...）

输入你要连接的服务器ip与用户

确认后会弹出一个新的会话框，文本框下方提示输入服务器的密码

好像需要两次输入，如图表示连接成功

现在就可以开始编写你的代码了，可以选择新建项目或者打开已有的目录

我这里选择新建一个文件（New file），ctrl+s进行保存的时候会弹出一个存储文件目录及名称的框进行选择

F5进行代码的调试运行，但是由于这是一个新的终端，可能之前在本地装的插件无法使用，需要重新添加插件，比如运行python脚本的插件（其他语言自行安装插件）

插件安装好，就可以进行调试运行了，F5选择调试环境文件

如图，成功运行py脚本
